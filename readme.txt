1. Технологический стек
- ASP.NET Core
- Angular 5 (VueJS)
- EF Core
- MongoDB
- Docker

1.1 Архитектура
- SPA приложение
- автоматизированное CI\DI + Docker

2. Основной функционал
- Создание любых опросов в админке. 
- Созданные модели опросов можно добавлять к своему боту Telegram
- Бот должен уметь задавать вопросы в чате. Сохранять результаты ответов.
